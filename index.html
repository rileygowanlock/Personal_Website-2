Just a little bit about me...

I am a second year Software Engineering student from the University of Waterloo. When my nose is not buried in a textbook, I like to spend my time playing guitar and ukelele. I am a passionate songwriter and performer.

Most importantly, I love to write code. I was introduced to programming at the age of 10. My older brother showed me a game he developed in his Computer Science class. It sparked an interest in me and in Grade 11 I took my first Computer Science course. I learned to code in Python and wrote many programs throughout my highschool career, including Duck Hunt, Pacman and Tetris.

It didn't take long to realize that I wanted to pursue a career in software development. I soon found myself sitting in a giant lecture hall, surrounded by other avid programmers. I learned C and C++ in first year and successfully implemented a compiler using C++ in second year.

After completing 8 months of school, I set off to get some real work experience. My program requires that I complete six 4-month co-op terms. For my first co-op, I worked at Gemalto in Ottawa. I developed a web-based client, and gained experience using HTML, CSS and Javascript with JQuery. I also had the opportunity to improve my Python skills, working on a Python GUI as well automating test coverage using the pytest framework.

At the beginning of this year I moved to Vancouver to complete my second co-op term. I worked as a fullstack developer at Coho Data. I wrote code to improve the Coho Data UI. I also designed and developed an interface for displaying server health statistics using Python and ReactJS.

Currently I am in Waterloo, finishing up second year.
